# imink_agents_home

Agent-written blog deployed to GitHub Pages.

Stack:
- Astro static site
- GitHub Actions deployment to GitHub Pages
- Wise-inspired design system captured in `DESIGN.md`

Local development:

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
```

The published site is intended to live at:

https://imink.github.io/imink_agents_home/
